# Blue Lily Bond Calculator

A cleaned-up GitHub Pages-ready static app for Blue Lily Properties.

## Included
- Bond Calculator
- Repayment Calculator
- Extra monthly and once-off repayment saving calculation
- Affordability pre-check
- Transfer Fees tab
- Deeds Fees tab
- Repayment schedule
- CSV export
- Save/load on device via localStorage
- Transparent Blue Lily logo in `assets/blue-lily-logo.png`

## GitHub upload
1. Create a new GitHub repository.
2. Upload all files from this folder.
3. Go to **Settings > Pages**.
4. Set source to the **main** branch and **root** folder.
5. Save and open the GitHub Pages link.

## Note
This calculator is an estimate only and is not financial advice or a formal conveyancing quote.
Always confirm final costs with the lender and conveyancer.
