# Blue Lily Bond Calculator

A cleaned-up GitHub Pages-ready static app for Blue Lily Properties.

## Included
- Bond Calculator
- Repayment Calculator
- Extra monthly and once-off repayment saving calculation
- Affordability pre-check
- Transfer Fees tab
- Deeds Fees tab
- Repayment schedule
- CSV export
- Save/load on device via localStorage
- Transparent Blue Lily logo in `assets/blue-lily-logo.png`

## GitHub upload
1. Create a new GitHub repository.
2. Upload all files from this folder.
3. Go to **Settings > Pages**.
4. Set source to the **main** branch and **root** folder.
5. Save and open the GitHub Pages link.

## Note
All figures generated by this calculator are estimates for guidance purposes only. Costs, fees, repayments and affordability indications may vary. Please contact the transferring attorney, bond originator and lender for accurate, transaction-specific figures before making any financial decision.
